## Payload

```json
{
  "songId": "3fa85f64-5717-4562-b3fc-2c963f66afa6",
  "title": "Explosive Groove",
  "artist": "FlowMart All Stars",
  "genre": "Jazz",
  "releaseDate": "2024-03-24"
}
```

- `songId` keeps the UUID that links SSE updates and enrichment workflows.
- `releaseDate` is optional; defaults to the ingestion timestamp when not provided.
